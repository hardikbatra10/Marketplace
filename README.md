🛒 Django Marketplace Web Application
A full-featured Marketplace Web Application built using Django, where users can register, upload products under categories, and chat with other users interested in buying those products.

🚀 Features
👤 User Authentication

User Sign Up

User Login / Logout

Secure authentication using Django's built-in auth system

📦 Product Management

Add new products

Upload product images

Assign products to categories

View product details

Dashboard to see products uploaded by the logged-in user

💬 Real-Time Chat (User-to-User)

Buyers can message sellers

One-to-one conversation system

Chat linked to specific products

📊 User Dashboard

View all uploaded products

Manage personal listings

Track interactions

🛠️ Tech Stack

Backend: Django

Database: SQLite (default) / PostgreSQL (optional)

Frontend: HTML, CSS, Bootstrap (if used)

Authentication: Django Authentication System
